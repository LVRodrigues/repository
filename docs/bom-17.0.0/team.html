O Time

Um projeto de sucesso requer muitas pessoas em diversos papéis. Alguns membros escrevem código ou documentação, enquanto outros testam o sistema, submetem atualizações ou sugestões.

O time é composto por Membros e Contribuidores. Membros tem acesso direto ao código fonte e desenvolvem o código. Contribuidores melhoram o projeto através do envio de atualizações e sugestões para os Membros. O número de Contribuidores para o projeto é ilimitado. Envolva-se hoje mesmo! Toda a contribuição é sempre bem vinda.
